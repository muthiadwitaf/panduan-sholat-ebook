
### Audio Generation
This project includes a `generate_audio.py` script that uses the `edge-tts` (Python) library to generate high-quality Arabic audio for the content.
The audio files are already generated in `assets/audio/`.

If you wish to regenerate or add more audio:
1.  Install Python.
2.  Run `pip install -r requirements.txt`.
3.  Edit `DATA` in `generate_audio.py`.
4.  Run `python generate_audio.py`.
